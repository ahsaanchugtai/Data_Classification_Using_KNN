# K-Nearest Neighbors (KNN) Classification Model

## Overview
This project demonstrates how to build a simple **Machine Learning Classification Model** using the **K-Nearest Neighbors (KNN)** algorithm in Python.

The program:
- Loads a dataset
- Splits the dataset into training and testing sets
- Trains a KNN classifier
- Makes predictions
- Evaluates model accuracy

This is a beginner-friendly machine learning project.

---

# Technologies Used

- Python 3
- Scikit-learn
- Pandas
- NumPy
